===================================================================================
                            INTERACTIVE ELEMENTS
===================================================================================

TOOLTIPS (appear on hover over ℹ️ icons):

  Overall Sentiment ℹ️:
  ┌────────────────────────────────────────────────────────────┐
  │ Composite sentiment score aggregated from news,            │
  │ analyst ratings, social media, and insider activity.       │
  │ Weighted by source reliability.                            │
  └────────────────────────────────────────────────────────────┘

  Sentiment by Source ℹ️:
  ┌────────────────────────────────────────────────────────────┐
  │ Breakdown of sentiment across different data sources.      │
  │ Each source is analyzed independently and weighted by      │
  │ reliability.                                               │
  └────────────────────────────────────────────────────────────┘

  Top Sentiment Drivers ℹ️:
  ┌────────────────────────────────────────────────────────────┐
  │ Key events, catalysts, and risks identified from recent    │
  │ news and market activity. These are the primary factors    │
  │ influencing current sentiment.                             │
  └────────────────────────────────────────────────────────────┘

===================================================================================
                           ANIMATION DETAILS
===================================================================================

PROGRESS BARS:
  • Entry animation: 0.5s ease-out from 0% to target%
  • Smooth gradient transitions on score changes
  • Pulse effect on high-confidence scores (>80%)

SENTIMENT ICON:
  • Fade-in on mount: 0.3s
  • Scale slightly on sentiment change: scale(1.1) → scale(1.0)
  • Color transition: 0.3s cubic-bezier(0.4, 0, 0.2, 1)

CARD HOVER EFFECTS:
  • Transform: translateY(-2px)
  • Shadow: Increase from md to lg
  • Duration: 0.3s ease

DATA FRESHNESS BADGE:
  • Updates every 60 seconds
  • Fade transition between states: 0.5s

===================================================================================
                         ACCESSIBILITY FEATURES
===================================================================================

KEYBOARD NAVIGATION:
  Tab Order:
    1. Overall Sentiment section
    2. Source Breakdown section (each source)
    3. Tooltip triggers (Tab to focus, Enter/Space to activate)
    4. Sentiment Drivers list items
    5. Market Context cards

SCREEN READER ANNOUNCEMENTS:
  • "Overall sentiment: Bullish, score 75 out of 100"
  • "News sentiment: 75%, 15 positive mentions, 3 negative"
  • "Analyst consensus: Buy, 12 buy ratings"
  • "Sentiment driver 1 of 5: Q4 earnings beat expectations"

ARIA LABELS:
  • aria-label="Sentiment panel"
  • aria-label="Overall sentiment gauge"
  • role="progressbar" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100"
  • aria-describedby="sentiment-tooltip-1"

COLOR CONTRAST:
  • All text meets WCAG AA standards (4.5:1 minimum)
  • Color is NOT the only indicator (icons + text + labels)
  • High contrast mode support via system preferences
